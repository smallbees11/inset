# linux最最基本的命令
- `ls`: 列出所在目录的文件
- `cd`：进入根目录
- `cd 目录名`：进入目录名/
-  `mkdir 文件夹的名字`: 新建文件夹
  

# vi基本命令
-`i`进入编辑模式
-`esc`退出编辑模式
-`：`shift+:进入控制模式
-`wq`:保存退出
-`w`:保存
-`q`:退出
-`dd`:删除一行

# git基本命令
- `git clone项目地址`
- `cd项目地址`
- `vim notes.md`
- `git add *`
- `git commit -m"注释"`
- `git push`
- `git diff`：查看提交了些什么
- `git log`:查看提交历史
- `git pull`：拉下来最新的代码
